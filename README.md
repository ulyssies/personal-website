## Overview

Designed and built without a framework or template. Features a 3D character sphere hero, particle starfield background, smooth scroll navigation, and live integrations with Spotify and Letterboxd. All sections — work, hobbies, and contact — built with vanilla HTML, CSS, and JavaScript.

---

## Sections

| Section | Description |
|---|---|
| **Hero** | 3D rotating sphere of characters with name and title |
| **About** | Background, tech stack badges, and bio |
| **Work** | Swipeable project carousel with live links and tech tags |
| **Hobbies** | Six hobby cards with descriptions |
| **What I'm Into** | Live Spotify playlist embed + Letterboxd profile card |
| **Contact** | Links to email, GitHub, LinkedIn, and resume |

---

## Features

- 3D sphere animation rendered on canvas
- Particle starfield rendered on canvas
- Scroll-triggered section animations
- Spotify embed with current playlist
- Letterboxd profile card
- Light/dark mode toggle
- Fully responsive layout

---

## Stack

Pure HTML, CSS, and JavaScript — no build tools, no dependencies, no framework.
